Static commercial website for Feed Me Chutney (feedmechutney.com) — a family-run artisan food brand handcrafting small-batch chutneys in Virginia's Northern Neck using locally sourced, farm-fresh ingredients with no preservatives.
The site features a full product showcase, embedded YouTube video gallery (13 videos), brand story section, social media integration, and direct links to the shop. Designed for mobile responsiveness with scroll-triggered animations, a dark/cream color scheme, and Playfair Display typography for an upscale editorial feel.
Stack: HTML5 · CSS3 · Vanilla JavaScript · Google Fonts · Font Awesome
Live site: feedmechutney.com
